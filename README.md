# شركة صقر للتدريب والاستشارات - منصة الأدوات الذكية

## نظرة عامة

منصة شاملة تجمع جميع أدوات الذكاء الاصطناعي والتعلم الآلي في مكان واحد، مقدمة من شركة صقر للتدريب والاستشارات.

## المميزات

- 🎯 **التحكم بالماوس بالعين**: تحكم في مؤشر الماوس باستخدام حركة عينيك
- 🤚 **التحكم بالماوس باليدين**: استخدم يديك للتحكم في الماوس والنقر
- 🔊 **التحكم في الصوت**: اضبط مستوى الصوت باستخدام حركة أصابعك
- 🔢 **عداد الأصابع**: تعرّف على عدد الأصابع المرفوعة في الوقت الفعلي
- 🖱️ **الماوس الافتراضي**: ماوس افتراضي متقدم مع تقنيات الذكاء الاصطناعي
- 🎤 **تحويل الكلام إلى نص**: حول كلامك إلى نص مكتوب
- 🔊 **تحويل النص إلى كلام**: استمع إلى النصوص بصوت طبيعي
- 🔍 **التعرف على الأشياء**: تعرّف على الأشياء في الصور
- 🌙 **كشف الحركة في الظلام**: اكتشف الحركة حتى في الظلام الدامس
- 📱 **مولد QR Code**: أنشئ رموز QR مخصصة بسهولة
- 🎮 **لعبة اليدين**: استمتع بلعبة تفاعلية تتحكم فيها بيديك

## المتطلبات

- Python 3.8 أو أحدث
- Django 4.2+
- OpenCV
- MediaPipe
- PyAutoGUI
- SpeechRecognition
- pyttsx3
- QRCode

## التثبيت والتشغيل

### 1. تحضير البيئة

```bash
# إنشاء مجلد للمشروع
mkdir saqr-training-system
cd saqr-training-system

# إنشاء بيئة افتراضية
python -m venv venv

# تفعيل البيئة الافتراضية
# على Windows:
venv\Scripts\activate
# على Linux/Mac:
source venv/bin/activate
```

### 2. تثبيت المتطلبات

```bash
pip install -r requirements.txt
```

### 3. تشغيل المشروع

```bash
# الانتقال إلى مجلد المشروع
cd saqr_training_system

# تشغيل الخادم
python manage.py runserver
```

### 4. الوصول للموقع

افتح المتصفح وانتقل إلى: `http://127.0.0.1:8000`

## استخدام الأدوات

### التحكم بالماوس بالعين

1. اجلس أمام الكاميرا بوضوح
2. انظر إلى النقطة التي تريد التحكم بها
3. أغمض عينك للنقر
4. تأكد من الإضاءة الجيدة

### التحكم بالماوس باليدين

1. ارفع إصبع السبابة للتحرك
2. قرب الإبهام والسبابة للنقر
3. حافظ على اليد في إطار الكاميرا
4. تأكد من الإضاءة الجيدة

### تحويل الكلام إلى نص

1. اضغط على "ابدأ الاستماع"
2. تحدث بوضوح وبصوت مناسب
3. تأكد من اتصال الميكروفون
4. امنح إذن الوصول للميكروفون

### تحويل النص إلى كلام

1. اكتب النص في المربع
2. اختر نوع الصوت المناسب
3. اضبط السرعة حسب الحاجة
4. اضغط على تشغيل للاستماع

### مولد QR Code

1. اكتب النص أو الرابط
2. اختر الحجم ومستوى تصحيح الخطأ
3. اضغط على "إنشاء QR Code"
4. حمّل أو اطبع النتيجة
