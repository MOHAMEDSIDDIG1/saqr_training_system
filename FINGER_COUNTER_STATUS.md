### 🎮 طريقة الاستخدام:

#### **1. تشغيل الخادم:**

```bash
python manage.py runserver
```

#### **2. الذهاب إلى الصفحة:**

```
http://127.0.0.1:8000/finger-counter/
```

#### **3. استخدام الأداة:**

1. **اضغط "بدء العد"** ▶️
2. **ستظهر نقاط اليد الصفراء** على الكاميرا
3. **ستظهر دائرة خضراء** على الإصبع السبابة
4. **ستظهر عدد الأصابع** في الزاوية العلوية

#### **4. اختبار اكتشاف اليد:**

1. **اضغط "اختبار اكتشاف اليد"** 🧪
2. **راقب النتائج** في مربع النتائج
3. **اتبع الاقتراحات** إذا لم تنجح

#### **5. تحسين الإعدادات:**

1. **اضبط "حساسية الكشف"**: 0.4 للبداية
2. **اضبط "تعقيد النموذج"**: 1 للبداية
3. **اضغط "تطبيق الإعدادات"** ⚙️
